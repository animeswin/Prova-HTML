Bem-vindo à Página Responsiva

Este é um exemplo de página HTML que utiliza CSS para criar um layout responsivo e aplicar uma paleta de cores definida por variáveis. As unidades flexíveis garantem que a página se adapte bem a diferentes tamanhos de tela.

Experimente redimensionar a janela do navegador para ver como o tamanho do texto muda conforme a largura da tela.
